# GBOX Achieves #1 Spot Among Pure Vision Agents on AndroidWorld

Date: 11 Oct 2025

Author: [Bazinga Wang](https://github.com/Bazinga-Wang)

We’re excited to share a big milestone from [GBOX](gbox.ai)!

By combining Claude Code agent with the GBOX MCP, we achieved an 86.2% success rate on the AndroidWorld benchmark, demonstrating GBOX’s strength in real-world Android automation. [View Full Report](https://github.com/babelcloud/android_world_benchmark/blob/main/GBOX/report.md).

![iamge](https://github.com/user-attachments/assets/2bb3ece9-ee14-44b2-a972-c35ad833c918)

## Why it matters:

- Natural language and vision-based controls make automation more robust and flexible.

- Claude Code can serve as a general-purpose agent with GBOX MCP, planning and execute tasks just like a human.
